# 👗 Vastra Elite

**Vastra Elite** is a modern and stylish fashion e-commerce web application designed to provide users with a seamless online shopping experience. The platform allows customers to explore fashion collections, browse products, view detailed product information, add items to their cart, and enjoy a smooth and responsive shopping experience.

## ✨ Features

* 🛍️ Browse a wide range of fashion and clothing products
* 🔍 Explore products by categories
* 📄 Detailed product pages
* 🛒 Add products to the shopping cart
* ➕ Increase or decrease product quantities
* ❌ Remove products from the cart
* 🔐 User authentication
* 🔑 Google Login integration
* 📱 Fully responsive design
* 🎨 Modern and minimal user interface
* ⚡ Fast and smooth user experience
* 🧭 Easy-to-use navigation

## 🛠️ Tech Stack

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone <your-repository-url>
```

### 2. Navigate to the Project

```bash
cd vastra-elite
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env` file and add the required environment variables.

```env
MONGO_URI=your_mongodb_connection_string
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```

> Never commit your `.env` file or sensitive credentials to GitHub.

### 5. Start the Development Server

```bash
npm run dev
```
